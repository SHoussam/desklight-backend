# desklight-backend

This is the backend for the DeskLight project. It provides API endpoints and database management for user authentication and other core features.

## Project Structure

- `api_root/` — Contains API endpoint logic (e.g., user management)
- `database/` — Database connection and models
- `securety.py` — Security and authentication logic

## Features
- User authentication and management
- Secure API endpoints
- Database integration

## Getting Started
1. Clone the repository
2. Install dependencies
3. Configure your environment variables and database
4. Run the backend server
